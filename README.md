# Last Game Foever

## Local LAN Multiplayer (Node.js + Socket.IO)

### Prerequisites
- Install **Node.js (LTS)** (comes with `node` + `npm`)

### Install
```bash
cd "Last Game Foever"
npm install
```

### Run
```bash
npm run dev
```

Then open the game from:
- `http://localhost:3000`

To test LAN:
- Player 1 clicks **Play** to create a room (room code shows on the character select screen)
- Player 2 clicks **Join** and enters the room code
- Both players pick a role and press **E** to Ready

## Controls
- **Move**: A / D
- **Jump**: W or Space
- **Attack**: Left mouse click
